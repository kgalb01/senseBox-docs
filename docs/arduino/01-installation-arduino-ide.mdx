# Installation der Arduino IDE

Bevor du die senseBox über die Arduino IDE programmieren kannst, musst du Treiber sowie eine Software auf deinem Computer installieren.

Schau dir die Anleitung für dein Betriebssystem an und folge den angegebenen Schritten.

  <TabsList>

  <TabsContent value="windows">
    <h2>Arduino Software für Windows herunterladen</h2>
    <p>
      Die senseBox ist ein Microcontroller mit verschiedenen Komponenten und
      Sensoren. Sie wird über die Entwicklungsumgebung _Arduino IDE_
      programmiert. Lade die neueste Version von der{" "}
      <a href="https://www.arduino.cc/en/software">Arduino Homepage</a>{" "}
      herunter:
    </p>
    <img
      src="/img/arduino-bilder/instalation/arduino-download.png"
      alt="arduino install"
    />
    <p>
      Arduino ist ein Open-Source-Projekt und wird durch Spenden finanziert.
      Daher wirst du vor dem Download nach einer Spende gefragt; falls du nichts
      spenden möchtest, kannst du auf `JUST DOWNLOAD` klicken.
    </p>
    <p>
      Führe den Installer aus und folge den Anweisungen. Die Installation ist in
      wenigen Minuten abgeschlossen.
    </p>
  </TabsContent>

    <p>
      Linux-Nutzer können das von Arduino bereitgestellte `AppImage` benutzen, um

    </p>
    <code>cd Downloads</code> <br />
    <code>chmod +x arduino-ide_2.2.1_Linux_64bit.AppImage</code> <br />
    <code>./arduino-ide_2.2.1_Linux_64bit.AppImage</code>
    <p>
