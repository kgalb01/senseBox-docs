# Installation

## Schritt 1: Vorbereitung
 - Entpacken Sie die Lieferung und überprüfen Sie den Inhalt auf Vollständigkeit.
 - Stellen Sie sicher, dass Sie sowohl die Halterung für die Sattelstütze als auch die für den Gepäckträger erhalten haben.

## Schritt 2: Montage der Halterungen

 - Wählen Sie den gewünschten Montageort für die Halterung aus: Sattelstütze oder Gepäckträger.
 - Befestigen Sie die Halterung an Ihrem Fahrrad. Achten Sie darauf, dass der kleine Hebel am Schlitten in Fahrtrichtung nach vorne zeigt.
 - Stellen Sie sicher, dass die Halterung sicher und fest angebracht ist, um ein Verrutschen während der Fahrt zu verhindern.


## Schritt 3 : Montage der senseBox:bike
 - Platzieren Sie die senseBox:bike auf dem Schlitten der Halterung.
 - Schieben Sie die senseBox:bike vorsichtig auf den Schlitten, bis sie sicher einrastet.
 - Sie können nun mit der Fahrt beginnen.

:::info
Hinweis: Bitte überprüfen Sie regelmäßig die Montage der senseBox:bike, um
sicherzustellen, dass sie sicher und korrekt angebracht ist.
:::


## Entfernen der senseBox:bike vom Fahrrad
Um die senseBox:bike zu lösen, drücken Sie einfach den kleinen Hebel am Schlitten, der sich in Fahrtrichtung vorne befindet. Durch Drücken des Hebels wird die
Verriegelung gelöst, und Sie können die senseBox:bike vom Schlitten abziehen. Achten Sie darauf, dies vorsichtig und kontrolliert durchzuführen, um Beschädigungen zu vermeiden.
