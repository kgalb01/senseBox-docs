# Software aufspielen und testen

Um die Ampel testen zu können, müsst ihr [hier](/docs/sensebox_co2_ampel_v1.bin) Programmdatei runterladen und auf die senseBox übertragen. Schließt dazu die Messstation mit dem USB-Kabel an einen PC
an. Neben dem USB-Anschluss auf der MCU befindet sich ein roter Knopf. Drückt ihn zweimal hintereinander.

![](/img/co2ampel/image20.jpg)

Daraufhin sollte in eurem Betriebssystem die senseBox als neuer Wechseldatenträger erscheinen.

![](/img/co2ampel/image21.png)

Kopiert jetzt die Datei _sensebox_co2_ampel.bin_ aus eurem Downloadordner auf den „SENSEBOX" Wechseldatenträger:

![](/img/co2ampel/image22.png)

Die Station startet danach automatisch neu und sollte euch die ersten Messwerte auf dem Display anzeigen.

![](/img/co2ampel/image23.jpg)

Für den Fall, dass das Display schwarz bleibt, drückt einmal auf den oben genannten Resetknopf, um das Programm auf der Station neu zu starten.
